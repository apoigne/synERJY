Check correctness of the installation:

0. If synERJY is installed for the first time restart the system

1. Copy the directory "target\win32\examples" of the synERJY directory
   to whereever you like

3. Launch synERJY

4. Set the workspace 
   - open preference panel Edit -> Preferences
   - click on ...
   - choose  directory "wherever-it-is"\examples
    
3. Load a project
   - File -> Load project, or click ... for project
   - select Basic1

3. Push the button "Build"

4. Push the button "Simulator"

Enjoy
